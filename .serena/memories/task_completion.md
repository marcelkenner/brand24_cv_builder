# Task completion checklist
Before considering a feature complete in this repo:
- add or update tests for changed behavior and run them,
- run lint and build (and type/test commands if configured),
- update README/docs to match actual behavior,
- keep files under the repo size limits,
- record assumptions explicitly when they cannot be eliminated,
- if working from an ExecPlan, update its `Progress`, `Decision Log`, `Surprises & Discoveries`, and `Outcomes & Retrospective` sections to reflect reality.
